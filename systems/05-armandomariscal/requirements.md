# Requirements: Personal Brand & Portfolio Hub

## Context
Este repositorio es el núcleo de la identidad profesional de Armando Mariscal.

## Functional Requirements (FR)
* **FR-01: Centralización de Proyectos:** Debe clasificar y exponer claramente los 7 proyectos clave del roadmap 2026.
* **FR-02: Visualización de Stack:** Exposición dinámica de tecnologías core (JavaScript, Vue, Django, Docker).
* **FR-03: Tracking de Actividad:** Integración de gráficas que demuestren consistencia en el desarrollo (Streaks y Activity Graph).
* **FR-04: Roadmap de Aprendizaje:** Declaración pública de objetivos actuales (DevOps, TypeScript Migration).

## Non-Functional Requirements (NFR)
* **NFR-01: UX/UI Limpia:** El README debe ser visualmente atractivo y fácil de navegar mediante tablas y badges.
* **NFR-02: Trazabilidad de Carrera:** La documentación debe reflejar la evolución de desarrollador fullstack.
* **NFR-03: Mantenibilidad:** El uso de plantillas Markdown debe permitir replicar la estructura de documentación en nuevos proyectos rápidamente.
* **NFR-04: Identidad Visual:** Uso de iconos y colores consistentes para identificar cada una de las 7 áreas de enfoque.
